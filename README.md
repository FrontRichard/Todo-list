# Todo List

Uma aplicação simples de lista de tarefas construída com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como Usar

1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.
